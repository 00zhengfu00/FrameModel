# FrameModel
###niko个人开发工具框架 
包括内容：
* Rxjava2.0+Retrofit2.0最新网络请求框架；
* 图片处理，运行时权限；
* 通用RecyclerVIewAdapter封装；
* activity和fragment封装；
* basePopWindow 封装。
* Realm 数据库

###运用项目 《青岛地铁电子检察平台》
